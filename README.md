# Volume Booster Extension

A sleek, private, and powerful browser extension that allows you to boost the volume of any HTML5 audio or video on a webpage up to 600%.

> **🎉 Now Available on Firefox Add-ons!**
>
> - [**Firefox Add-ons (AMO)**](https://addons.mozilla.org/en-US/firefox/addon/volume-booster/) ✅ **Official Release**
> - [**Chrome Web Store**](#chrome-installation-guide) - Install from source (free alternative)
> - **Edge, Brave, Vivaldi** - Install from source (Chromium-based)

## ✨ Features

- **Boost Volume up to 600%**: Break past the default 100% volume limits on quiet videos or calls.
- **Beautiful & Modern UI**: A clean, dark-mode focused neon user interface.
- **Cross-Browser Compatible**: Works perfectly on Google Chrome, Mozilla Firefox, Edge, and other Chromium-based browsers (Manifest V3).
- **100% Privacy Focused**: The extension does **not** collect, store, or transmit any user data or telemetry. Your browsing habits remain entirely on your device.

## 🚀 Installation & Usage

### Firefox Installation (Official Store) ⭐

**The easiest way to install Volume Booster!**

1. Visit [Volume Booster on Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/volume-booster/)
2. Click the **"Add to Firefox"** button
3. Click **"Add"** when prompted
4. Open any video and use the extension icon to boost volume!

---

### Chrome Installation Guide

Since we can't afford the $5 Chrome Web Store developer fee right now, here's how to install it for free on Chrome, Edge, Brave, or any Chromium-based browser:

#### Method 1: Load from Source (Recommended - 2 minutes)

1. **Clone or Download** this repository to your computer
2. Open your browser and go to `chrome://extensions/` (or equivalent for your browser)
3. Enable **"Developer mode"** toggle in the top right corner
4. Click **"Load unpacked"**
5. Navigate to and select the `chrome` folder from this project
6. Done! Pin the extension to your toolbar
7. Open any video (YouTube, Netflix, etc.) and click the Volume Booster icon to adjust volume

#### Method 2: Using the ZIP File

1. Download this repository as a ZIP file
2. Extract the ZIP to a folder on your computer
3. Open `chrome://extensions/` in your browser
4. Enable **"Developer mode"**
5. Click **"Load unpacked"** and select the extracted `chrome` folder
6. Start boosting!
